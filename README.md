# MegaCasting

Ce projet a pour but de gérer les activités de la société MegaProduction.

  - Gestion des offres
    - Graphique du nombre d'offres en fonction du temps
  - Gestion des clients
  - Gestion de la localisation
  - Gestion des contrats
  - Personnalisable
